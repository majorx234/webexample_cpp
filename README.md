# First aproach Boost Beast
## Build
``` bash
git clone https://github.com/majorx234/webexample_cpp.git
cd webexample_cpp
mkdir build && cd build
cmake ..
make
```

## Run & Test
- run ```./web_server 127.0.0.1 31337 100 8```
- test: ```curl 127.0.0.1:31337/test```
